HTML

HTML & CSS Questions

1
What is semantic HTML and why is it important?
Semantic HTML uses tags that describe their meaning (like <header>, <article>), improving SEO and accessibility.
2
What's the difference between block, inline, and inline-block elements?
Block: takes full width; Inline: fits content; Inline-block: inline but can set width/height.
JS

JavaScript Questions

1
What's the difference between var, let, and const?
var: function-scoped, hoisted; let: block-scoped, mutable; const: block-scoped, immutable.
React

React Questions

1
What is the virtual DOM and how does it work?
Virtual DOM is a JavaScript representation of the real DOM that React uses to efficiently update the UI.
Node

Node.js Questions

1
What is Node.js and how does it work?
Node.js is a JavaScript runtime built on Chrome's V8 engine that allows JavaScript to run on the server side.

Express.js Questions

What is Express.js and why is it used?
Express.js is a minimal and flexible Node.js web application framework that provides robust features for web applications.

What is MongoDB and what are its advantages?
